# Project Name: Ian Mount's Portfolio

## Description
Ian Mount's Portfolio is a showcase of his recent projects as a junior web developer based in Chicago, IL. The portfolio provides visitors with a curated selection of his work and a brief introduction about himself.

## Features

- **Responsive Design:** Ensures optimal viewing experience across various devices.
- **Dynamic Content:** Portfolio content is fetched dynamically from a data source using Astro's content fetching capabilities.
- **Project Showcase:** Ian's recent projects are displayed in an organized manner for visitors to explore.
- **Contact Call-to-Action:** Includes a section for contacting Ian, facilitating potential collaborations or inquiries.
- **Styled Components:** Various components and sections of the portfolio are styled using CSS for enhanced visual appeal and layout.

## Technologies Used

- **Astro:** Utilized for building the static site and managing components.
- **JavaScript:** Used for dynamic content fetching and interaction.
- **CSS:** Styling components and sections for enhanced visual presentation.

## Installation

To run the project locally, follow these steps:

1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd ian-mount-portfolio`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`
5. Open your browser and visit [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Components Of Note

### ContactCTA Component
This component provides a call-to-action section for visitors to get in touch with Ian. It includes links to his GitHub and LinkedIn profiles, as well as a contact form for sending messages.

### Contact Form
The contact form allows visitors to send messages directly to Ian. It includes fields for name, email, and message, along with a submit button.
